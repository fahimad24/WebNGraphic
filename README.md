## 📖 About / Description
**WebnGraphic** is a comprehensive, modern, and highly interactive web application designed for an IT services and engineering firm. It serves as a unified digital platform to showcase premium services, engage clients with stunning designs, and streamline the customer pipeline. 

## 🛠️ Technology Stack (Tech Use Korechi)
This project is built using cutting-edge, modern web technologies:

- **Framework**: [Next.js 16](https://nextjs.org/) (App Directory)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Animations & 3D**: [Framer Motion](https://www.framer.com/motion/) & [Three.js](https://threejs.org/) / React Three Fiber
- **Database**: PostgreSQL with [Prisma ORM](https://www.prisma.io/)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/) (Auth.js)
- **Forms & Validation**: React Hook Form, Zod, and Google ReCAPTCHA v3
- **Email & Communications**: Nodemailer (with Brevo)
- **Analytics**: Google Analytics

## 🎯 What it does (Ki Kaj Kore?)
This platform essentially solves several business needs natively:
1. **Service Showcasing:** Beautifully highlights services like CCTV Solutions, IT Support, Networking Solutions, and Software Development with engaging gradients and micro-animations.
2. **Client Booking System:** Features a built-in calendar scheduling tool for clients to book consultations or services.
3. **Secure Contact Processing:** Uses invisible ReCAPTCHA and Nodemailer to handle client inquiries efficiently, blocking bots automatically.
4. **Admin Management Dashboard:** A protected route for administrative tasks involving forms, bookings, and site content.

## 🤔 Why it is built (Keno Banano Hoyeche?)
This project was built to transform a standard service business into a premium digital brand. 
It’s crafted to:
- Establish absolute trust and authority in the IT & Engineering space.
- Provide a smooth, "wow-factor" experience using dark mode aesthetics, glassmorphism, and smooth scroll animations.
- Automate interactions so the business operates efficiently online.
